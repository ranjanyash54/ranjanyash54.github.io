Simulators like SUMO are high-fidelity but slow, which limits how much they can be used downstream. Enactor addresses this by learning a fast, transformer-based surrogate world model that reproduces simulator dynamics at a fraction of the computational cost.

The framework takes an actor-centric approach inspired by World Model paradigms, jointly modeling vehicle and pedestrian trajectories with cross-interactions and uncertainty-aware (Gaussian) predictions. Training uses a simulation-in-the-loop methodology, where the model rolls itself forward using its own predictions rather than ground-truth data, ensuring physical consistency across full trajectories rather than just single-step accuracy.

Evaluated over 40,000 simulation timesteps, Enactor generates long-horizon, physically consistent trajectories using significantly fewer training samples than traditional agent-centric generative approaches, outperforming baselines by over 10x on KL-Divergence metrics for both traffic-specific and aggregate measures. A real-time inference server (deployed via ZeroMQ) and an open dataset are released for reproducibility.
